Using the Phone Book

This mini app lets you add, search, edit, delete, and back up contacts in your browser. Data is saved in localStorage.

Add a contact

  1. On the Contacts page, select Add Contact.
  2. Fill in Name and Phone. Email and Notes are optional.
  3. Select Save.

Search

Type in the search box to filter by name, phone, or email. Clear resets the list.

Edit or Delete

Select Edit to update fields, or Delete to remove the contact.

Backup and Restore

Open Preferences. Use Export to download a JSON file. Use Import to choose a JSON file and merge contacts. If Auto-save Backup is enabled, a download triggers after changes.

FAQ

  • Where is my data stored? In your browser localStorage on this device.
  • Will I lose data on refresh? No, localStorage persists. Clearing browser data removes it.
  • Can I move data to another device? Export JSON on this device and Import it on the other.
